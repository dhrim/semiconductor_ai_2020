# 교육 목표

딥러닝을 사용한 반도체 데이터 분석 방법을 이해한다.


<br>

# 교육 상세 목표

- 딥러닝의 개념을 이해한다.
- Keras로 구현한 딥러닝 코드를 파악하고 이해한다.
- 딥러닝으로 해결할 수 있는 반도체 문제를 이해한다.
- 딥러닝에 적용하기 위한 반도체 데이터를 이해한다.
- 실제 반도체 데이터에 딥러닝을 적용하여 결과를 도출한다.


<br>

# 대상
